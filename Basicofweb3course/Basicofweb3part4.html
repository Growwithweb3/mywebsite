Introduction

The marketing tool of web 3 where project to attract user and to create hype in market is called airdrop. User can contribute in project either by project testing (testnet, mainnet, and node run) or socifi (galxe, kaito…) and later project reward with their crypto, NFT or some time USDT. Think as it like you are helping projects in bug finding by doing testnet, promotion of socila media by posting content related to them and for that project is giving you some token. if project perform well in the market then you are rich as fuck. sometime project announce airdrop camping but later scam their communit.
